# ramble

ローカルで動く音声文字起こしツール。Apple Silicon Mac + mlx_whisper で、10-15分の長時間音声をサッと文字起こしする。

## 何のために作ったか

Claude CodeやClaude Desktopの音声入力は短い時間しか受け付けられない。整理されていなくてもとにかく喋りまくってアウトプットし、それを後から生成AIに整理させてコンテキストとして食わせたい、というワークフローのために作った。

## ワークフロー

```
喋る → 生テキスト出力 → 生成AIで整理 → AIコンテキストとして利用
        (このツール)      (別途)
```

文字起こし結果はフィラー（えーと、あのー等）を含む生テキストで出力する。整理・要約・構造化は別途生成AIに任せる設計。

## セットアップ

Apple Silicon Mac (M1以降) が必要。

```bash
pip install mlx-whisper sounddevice numpy
```

初回の文字起こし時にWhisperモデルが `~/.cache/huggingface/hub/` にダウンロードされる（large-v3-turboで約1.5GB）。2回目以降はキャッシュから読み込むためネットワーク不要。

## 使い方

```bash
python ramble.py
```

起動するとメニューが表示される:

```
==================================================
  ramble
  マイク: SPACE
  モデル: large-v3-turbo
==================================================

  [Enter] 録音開始
  [d]     マイクデバイス変更
  [m]     モデル変更
  [q]     終了
```

- **Enter**: 録音開始。もう一度EnterまたはCtrl+Cで停止し、文字起こしが始まる
- **d**: マイクデバイスを切り替え（外部マイク/内蔵マイク等）
- **m**: Whisperモデルを切り替え（large-v3-turbo / medium / tiny）
- **q**: 終了

文字起こし結果は:
- ターミナルに表示
- クリップボードにコピー（pbcopy）
- `output/` ディレクトリにタイムスタンプ付きテキストファイルとして保存

## リソース消費

| 状態 | CPU | メモリ |
|------|-----|--------|
| 待機中（メニュー表示） | ほぼ0 | モデル常駐分のみ (数GB) |
| 録音中 | 微小 | 録音データ (15分で約28MB) |
| 文字起こし中 | GPU/ANE活用 | モデル + 一時処理用 |

録音データとテキストは毎回解放されるため、起動しっぱなしでもメモリが膨らむことはない。モデルはプロセス終了まで常駐するが、2回目以降の文字起こしが高速になるメリットがある。
